# Intallations

Setup a conda environment

```
conda create --name po-pandas
source activate po-pandas
```

Get jupyterlab up and running

```
conda install -c conda-forge jupyterlab
```

Launch jupyter lab

```
jupyter lab
```

## Pandas

```
conda install pandas scipy
```

## Vaex

```
conda install vaex
```

## Dask
Do a powered with dask thing. 

```
conda install dask
```
